Objectifs

Dans cet TP, vous allez enregistrer quelques objets ou lieux qui pourraient utiliser des capteurs pour collecter des données et déterminer si les données collectées sont sensibles.

Contexte / scénario

Il est important de reconnaître les endroits où des capteurs sont utilisés dans notre monde actuel et les types de données qu'ils collectent. Nous devons déterminer si les données collectées sont de nature sensible. S'il s'agit d'informations personnelles sensibles ? À qui pourrait profiter un vol ?

Instructions

Sélectionnez 3 ou 4 objets/lieux « inhabituels » qui pourraient maintenant contenir des capteurs. Répertoriez les types de données que les capteurs pourraient collecter. Déterminez si certaines des données collectées sont sensibles. Enregistrez vos résultats dans un tableau comme ci-dessous.
