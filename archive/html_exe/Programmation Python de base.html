Arrière-plan

Python est un langage de programmation qui permet de créer des instructions simples. Python est très simple d'emploi, puissant et polyvalent. Il est devenu le langage préféré de nombreux développeurs IoT. La communauté de développeurs est à l'origine de la popularité du langage Python. Les développeurs Python ont en effet créé et mis à disposition de nombreux modules spécifiques pouvant être importés dans tous les programmes afin d'offrir immédiatement des fonctionnalités supplémentaires.

Scénario

Au cours de ces travaux pratiques, vous apprendrez des notions de base sur la programmation Python et vous les mettrez en pratique. Nous utiliserons Python version 3 dans ces travaux pratiques.

Ressources requises

=  Un ordinateur personnel moderne avec accès à Internet et suffisamment de mémoire vive.

=  Python3 / IDLE3

Instructions

Partie 1 : Notions de base sur Python

Dans cette partie, vous apprendrez des notions de base sur la programmation Python.

Étape 1: Téléchargez et installez Python3.

a.  Rendez-vous sur www.python.org/downloads/ et téléchargez la dernière version pour votre système d'exploitation.

b.  Localisez le fichier téléchargé et installez Python. Sélectionnez Add Python 3.x to PATH, où x correspond à la version. L'ajout de Python au chemin indique à Windows où vous avez installé Python. Cliquez ensuite sur « Installer ».

Vous pouvez également installer Python 3.x via le Microsoft Store. Recherchez Python et sélectionnez la dernière version de Python.

Python est déjà installé sur la plupart des distributions Linux. Pour installer IDLE sous Ubuntu, saisissez la commande suivante :

user@Ubuntu:~$ sudo apt-get install idle3

Pour iOS, consultez la page https://www.python.org/download/mac/tcltk/.

Étape 2: Programmation Python dans l'interpréteur interactif.

Comme Python est un langage interprété, il est possible de lancer ses commandes dans un interpréteur interactif.

Les étapes de cet exercice concernent l'exécution de Python 3 sous Windows.

a.  Démarrez Python3. (Sous Windows, cliquez sur Démarrer > Python3.x > Python3, où x correspond à la version.)

Python 3.10.1 (balises/v3.10.1:2cd268a, 6 déc. 2021, 19:10:37) [MSC v.1929 64 bits (AMD64)] sur win32

Tapez "help", "copyright", "credits" ou "license" pour plus d'informations.

>>> 

b.  Effectuez les calculs. Saisissez les calculs à l'invite Python >>>.

>>> 1 + 2

3

>>> 2 * 4

8

>>> 6 / 2

3.0

c.  Imprimer une chaîne de texte.

>>> ‘Comment allez-vous?’

‘Comment allez-vous ?’

d.  Utilisez la commande type() pour déterminer le type de donnée de base : entier, flottant, chaîne, booléen.

>>> type(65)

<class 'int'>

>>> type(45.6)

<class 'float'>

>>> type("Hi!")

<class 'str'>

>>> type(True)

<class 'bool'>

>>> 1<2

True

>>> 1<1

Faux

>>> 1==1

True

>>> 1>=1

True

e.  Créer une variable.

>>> x=3

>>> x*5

15

>>> "Bon!"*x

'Bon !Bon !Bon !'

f.   Associer plusieurs chaînes et les imprimer comme une seule.

>>> str1="Cisco"

>>> str2="Networking"

>>> str3="Academy"

>>> space=" "

>>> print(str1+space+str2+space+str3)

Cisco Networking Academy

g.  Convertir le type de données de nombre en chaîne.

>>> x=5

>>> str(x)

‘5’

>>> y=4.2

>>> str(y)

‘4.2’

h.  Notez que les entiers ne sont pas arrondis lorsqu'ils sont convertis depuis le type flottant. La décimale est ignorée.

>>> int(8.21)

8

>>> int(8.99)

8

>>> int(8.21) + int(8.99)

16

i.  Convertir un nombre entier en nombre flottant.

>>> x=5

>>> x

5

>>> float(x)

5.0

>>> type(x)

<class 'int'>

>>> x=float(x)

>>> type(x)

<class 'float'>

>>> x

5.0

j.  Obtenir l'entrée de l'utilisateur.

>>> name=input("What is your name? ")

Quel est votre nom ? John

>>> print("Bonjour " + John + ", ravi de vous rencontrer")

Bonjour John, ravi de vous rencontrer !

k.  Utilisez la commande quit() pour quitter l'interpréteur interactif.

Partie 2 : Créer un script avec IDLE pour Python

Dans cette partie, vous allez lancer IDLE et créer un script simple. IDLE est l'acronyme de Integrated Development and Learning Environment. Il est pris en charge et inclus dans le package Python. Voici quelques-unes des fonctions clés de IDLE pour Python :

=  Une fenêtre de shell Python (interpréteur interactif) avec colorisation de l'entrée et de la sortie de code, ainsi que des messages d'erreur.

=  Un éditeur de texte multifenêtre avec niveaux d'annulation multiples, colorisation Python, indentation intelligente, conseils d'appel, saisie semi-automatique et d'autres fonctionnalités.

=  Capacité de rechercher dans n'importe quelle fenêtre, de remplacer dans les fenêtres d'éditeur et de rechercher dans plusieurs fichiers (grep).

=  Un débogueur avec points d'arrêt persistants, pas-à-pas et affichage des espaces de nom globaux et locaux.

=  Configuration, navigateurs et autres boîtes de dialogue.

Étape 1: Lancez IDLE.

a.  Démarrez IDLE dans le menu Démarrer de Windows.

b.  Cliquez sur File -> New File pour ouvrir un nouveau script Python (sans titre).

c.  Saisissez le code du script print(“Hello World!”), notez que les codes sont en couleur et que les parenthèses d'ouverture et de fermeture correspondent.

d.  Cliquez sur Fichier > Enregistrer, enregistrez le script actuel sous 1.py dans votre répertoire Documents (C:/Users/your name/Documents). Cliquez sur le bouton Save.

e.  Cliquez sur Run -> Run Module (ou appuyez sur F5). La fenêtre de shell affiche le résultat.

Étape 2: Créez votre propre script.

Vous ne savez peut-être pas comment créer vous-même un script Python complexe. Mais vous pouvez créer un script Python simple en utilisant les exemples de la partie précédente.

Par exemple, créez et enregistrez le nouveau fichier en tant que 2.py dans votre répertoire Documents. Saisissez print(1+1) dans le nouveau fichier et enregistrez-le à nouveau. Exécutez le module et imprimez les résultats de 1+1 et affichez-les dans le shell.

Testez différentes fonctions et vous serez sur la bonne voie pour utiliser un langage de programmation puissant.
