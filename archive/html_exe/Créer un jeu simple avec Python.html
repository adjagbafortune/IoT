Ressources requises

=  Un ordinateur personnel moderne disposant de suffisamment de RAM et d'un accès Internet.

=  Python/IDLE installés.

Instructions

L'objectif du jeu est de trouver un nombre choisi par un joueur entre 0 et 1 024 en utilisant la méthode de bissection.

Partie 1 : Créer un jeu simple avec Python IDLE

a.  Lancez l'activité IDLE. (Sous Windows, cliquez sur Démarrer > Python3.x > IDLE, lorsque x est la version finale.)

b.  Créez un fichier dans IDLE. Cliquez sur File -> New File pour ouvrir un nouveau script Python (sans titre).

c.  Copiez et collez le script suivant dans le nouveau fichier.

x=entier(y)

a=0

d.  Cliquez sur Fichier > Enregistrer, enregistrez le script actuel sous le nom game.py dans votre répertoire C:/Users/nom d'utilisateur/Documents. Cliquez sur Enregistrer (Save) pour continuer.

e.  Cliquez sur Run -> Run Module (ou appuyez sur F5). La fenêtre de shell affiche le résultat.

f.   Si une erreur se produit pendant l'évaluation de la syntaxe du code, corrigez-la.

Partie 2 : Remarques générales

1.  Comment détecter si le joueur saisit un nombre qui n'est pas compris entre 0 et 1 024 ?

Zone de réponse

Vous devez vérifier que la saisie est comprise dans la plage appropriée immédiatement après que le joueur a saisi son nombre. Une autre instruction if peut être ajoutée pour tester si le nombre est compris entre 0 et 1024.

2.  Comment détecteriez-vous qu'un joueur saisit un nombre flottant ?

Zone de réponse

Vous devez vérifier que la saisie correspond à un nombre entier immédiatement après que le joueur a saisi son nombre. L'entrée utilisateur est une chaîne. Vous devez donc vérifier qu'il ne s'agit que de valeurs numériques dans la chaîne d'entrée. Par exemple, y.isnumeric () peut tester si l'entrée de l'utilisateur ne contient que des chiffres numériques.

Voici un exemple de code :

y=input('Saisissez un entier compris entre 0 et 1024 -- ')
a=0
b=1024
test=True

while test == True:
    if y.isnumeric():
        x=int( y)
        if x == 0:
            print('Votre numéro est 0, merci d'avoir joué.')
            test=False
        elif x == 1024:
            print('Votre numéro est 1024, merci d'avoir joué.')
            test=False
        elif x > 1024 or x < 0:
                y=input('Saisissez un entier entre 0 et 1024 -- ')
                test=True
        else:
            n=int((a+b)/2)
            if n == x:
                break
            else:
                if n < x:
                    a=n
                else:
                    b=n
            test=True
    else:
        y=input('Saisissez un entier compris entre 0 et 1024 -- ')
        test=True

print('Your number is ', n, ', Merci d'avoir joué.')
